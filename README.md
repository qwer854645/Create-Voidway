# Create: Voidway

A Create addon for Minecraft **1.21.1** (NeoForge) that extends [Create Utilities](https://github.com/Duqueeee/create-utilities) with void logistics, portals, and teleport pads.

## About

Create: Voidway lets you move stress, items, fluids, and energy across distances using frequency-linked void devices. This fork builds on the unofficial NeoForge port by SmartStreamLabs and adds split input/output blocks, void transfer fluid as an operating catalyst, teleport pads, and configurable void portals.

## Features

- **Void motors, chests, tanks, and batteries** — wireless-style transfer over a shared frequency channel
- **Dedicated input and output blocks** — clearer logistics for stress, items, fluids, and FE
- **Void transfer fluid** — consumed by many void devices while they run
- **Void teleport pads** — instant entity transport between paired pads
- **Void portals** — frame-based portals fueled by stress and void transfer fluid
- **Gearcube and L-shaped gearbox** — compact kinetic routing from the original mod
- **Ponder tutorials** — in-game guides for all major blocks

## Requirements

- Minecraft `1.21.1`
- NeoForge `21.1.x`
- Create `6.0.9+`

## Build

```bash
./gradlew build
```

Run the client:

```bash
./gradlew runClient
```
